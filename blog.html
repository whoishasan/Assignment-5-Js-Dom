Important FAQ

What is the Document Object Model (DOM)?

The DOM is a structured representation of a webpage, allowing JavaScript to interact with elements.

How do you select an element from the DOM?

You can select an element from the DOM using methods like:

  • document.getElementById("id")
  • document.querySelector("selector")
  • document.getElementsByClassName("className")
  • document.getElementsByTagName("tagName")
What is event delegation in the context of the DOM, and why is it useful?

Event delegation allows a parent element to manage events from its child elements, improving performance by reducing multiple event listeners and enhancing memory usage efficiency.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute, getAttribute, and removeAttribute. For styles, modify the style property directly, like element.style.color = "red" or add/remove CSS classes using classList.add() and classList.remove().
